Notion 拥有非常完善的快捷键系统，你不需要使用鼠标就几乎可以执行所有的操作🐭

注解：你会发现大量的用到了 cmd/ctrl 。 它们分别对应的是 Mac上的 command 键和Windows、Linux上的 control 键。 这份文档是针对 English/QWERTY 键盘下的快捷键。

最常用

记住以下几个命令，可以让你的工作非常的快速：

  • cmd/ctrl + n 新建一个页面（仅适用于桌面客户端）。
  • cmd/ctrl + shift + n 或者 cmd/ctrl + click 打开一个新的 Notion 窗口
  • cmd/ctrl + p 打开搜索或者跳到最近打开页面。
  • cmd/ctrl + [ 回上一个页面。
  • cmd/ctrl + ] 去下一个页面。
  • cmd/ctrl + shift + l 切换到黑夜模式 (这是 L 不是 1)。

小建议：用你电脑自带的 emoji picker 来添加行内 emoji 元素，下面是打开 emoji picker 的快捷方式：

Mac: ctrl + cmd + space

Windows 10: windows key + . or windows key + ;

Markdown 快捷键

行内快捷方式:

  • 在文本两侧使用 ** 创建 粗体。
  • 在文本两侧使用 * 创建 斜体。
  • 在文本两侧使用 ``` 创建 行内代码块。(这个按键在数字 1 的左边。)
  • 在文本两侧使用 ~ 创建 删除线文字。

在一行的开头尝试下面的快捷方式：

  • 输入 *, -, 或者 + 后再输入 空格 创建一个无序列表。
  • 输入 [] 创建一个待办事项。（在中间不需要输入 空格 。）
  • 输入 1. 后再输入 空格 创建一个有序列表。
  • 输入 # 后再输入 空格 创建一个 H1 标题。
  • 输入 ## 后再输入 空格 创建一个 H2 标题。
  • 输入 ### 后再输入 空格 创建一个 H3 标题。
  • 输入 > 后再输入 空格 创建一个下拉列表。
  • 输入 " 后再输入 空格 创建一个引用块。

内容样式编辑

  • enter 在下方新建一个 block 。
  • shift + enter 仅仅换行，而非新建一个 block 。
  • cmd/ctrl + shift + m 新建一条评论。
  • --- 创建分割线。(连续的三个破折号。)
  • 选中文本后按 cmd/ctrl + b 创建 粗体。
  • 选中文本后按 cmd/ctrl + i 创建 斜体。
  • 选中文本后按 cmd/ctrl + shift + s 创建 删除线文字。
  • 选中文本后按 cmd/ctrl + k 创建带链接的文字。你也可以复制 URL 后选中文字直接按 cmd/ctrl+v.
  • 选中文本后按 cmd/ctrl + e 创建 行内代码块。
  • tab 缩进。提示：缩进会根据 father block 进行改变。
  • shift + tab 取消缩进。
  • 在一行的结尾或开头处输入 /turn 可以把当前行改为其它 block 。
  • 在一行的结尾或开头处输入 /color 可以改变当前 block 的文字颜色。（输入 /default 可以将颜色改回默认。）常用颜色设置： /blue, /blue background

  • cmd/ctrl + option/shift + 0 创建一个文本。

  • cmd/ctrl + option/shift + 1 创建一个 H1 标题。

  • cmd/ctrl + option/shift + 2 创建一个 H2 标题。

  • cmd/ctrl + option/shift + 3 创建一个 H3 标题。

  • cmd/ctrl + option/shift + 4 创建一个待办事项。

  • cmd/ctrl + option/shift + 5 创建一个无序列表。

  • cmd/ctrl + option/shift + 6 创建一个有序列表。

  • cmd/ctrl + option/shift + 7 创建一个下拉列表。

  • cmd/ctrl + option/shift + 8 创建一个代码块 - 例如：

    Any kind of code can go here and be copied to your clipboard!

  • cmd/ctrl + option/shift + 9 把当前行变为 page 页。

  • cmd/ctrl + + 放大显示。

  • cmd/ctrl + - 缩小显示。

  • 选中内容后按住 option/alt 拖拽可以复制这些内容。

编辑/移动 blocks

在 Notion 里万物皆是 block ，这些快捷键适用于你选中 block 后的操作。

  • esc 选中或取消选中你所在的 block 。
  • cmd/ctrl + a 选中光标所在 block 的所有内容。
  • space 全屏显示你选中的图片或者取消全屏显示你选中的图片。
  • arrow keys 选择不同的 block 。（翻译官注：上下左右键）
  • shift + up/down arrow keys 选中当前 block 里光标往前或往后的全部内容。
  • 使用 Mac 按 cmd + shift + click ，使用 Windows/Linux 按 option + shift + click 可以跨行选中 block 。
  • shift + click 在无光标的情况下可以选中任意 block ，在光标的情况下可以选中光标所在 block 相邻的 block 。
  • backspace 或者 delete 删除选中的 block 。
  • cmd/ctrl + d 复制光标所在 block 。
  • enter 编辑选中 block 的文字。（或者在当前页面打开一个弹窗 page ）
  • cmd/ctrl + / 编辑或改变选中的 block 。
  • 在 board 视图中，选中多个卡片后按 cmd/ctrl + / 可以一次性编辑多个任务的状态。
  • 按住 cmd/ctrl + shift + arrow keys 移动选中 block 的位置。
  • cmd/ctrl + option/alt + t 展开或者关闭下拉列表。
  • cmd/ctrl + shift + h 对选中文本应用上一个颜色设定。
  • cmd/ctrl + enter 对当前块进行快速操作。快速操作包括：
    • 打开 page
    • 修改待办事项的样式
    • 展开或者关闭下拉菜单
    • 全屏或者取消全屏展示图片

@ 命令的用法

  • 提到一个用户 - ****输入 @ + 另一个用户的名称可以让该用户获得通知提醒。这个方法可以用在评论和讨论中。
  • 链接到一个 page - 输入 @ + 一个页面的名称可以创建链接到该页面的 block 。如果你改变了页面的名称，这个 block 链接的名称也会自动改变。
  • 提到一个日期 - 输入 @ + 一个任意格式的时间（例如："yesterday," "today" 、"tomorrow," 或者 "next Wednesday")。
  • 添加一个提醒事件 - 输入 @remind 后跟着任意的时间（包括 "yesterday," "today," "tomorrow," 等等。）。您可以单击出现的链接来调整您想要提醒的日期和确切时间。当时间到来时你会收到一个提醒。点击查看更多与提醒有关的内容→
  • 当你仅仅是想要输入一个 @ 符号而不是想要使用 @ 命令的时候，在输入 @ 后按下 esc 。

/ 命令的用法



在 Notion 输入 / 会显示一个完整的 block 操作菜单。在输入 / 后接着输入下面的内容可以进行相对应的操作：

基础

  • /text 或者 /plain 创建一个文本。
  • /page 创建一个 page 。（如果你按下 enter ，这个 page 将被打开）。
  • /bullet 创建一个无序列表。
  • /num 创建一个有序列表。
  • /todo 创建一个待办事项。
  • /toggle 创建一个下拉菜单。
  • /div 创建一个分割线。
  • /quote 创建一个引用块。
  • /h1 或者 /# 创建一个 H1 标题。
  • /h2 或者 /## 创建一个 H2 标题。
  • /h3 或者 /### 创建一个 H3 标题。
  • /link 创建一个可以跳转到其它 page 的链接。
  • esc 取消 / 命令。如果你只是想要输入一个 / 符号，而不是一个 / 命令。

数据表格

  • /table-inline 在当前页面创建一个表格。
  • /board-inline 在当前页面创建一个看板。
  • /calendar-inline 在当前页面创建一个日历。
  • /list-inline 在当前页面创建一个列表。
  • /gallery-inline 在当前页面创建一个画册。
  • 可以 -full 代替 -inline 创建一个带相对应视图的 page 而不是在当前页面创建。例如：/table-full, /board-full
  • /linked 创建一个可以链接到其它 page 数据表格的表单副本。

多媒体

  • /image 可以从本地、超链接地址或者 Unsplash 上添加图片。
  • /pdf 可以从本地、超链接地址上添加 PDF 文档。
  • /book 输入链接来创建一个书签。如下：

Notion - The all-in-one workspace for your notes, tasks, wikis, and databases.

  • /video 可以从本地或者 YouTube、Vimeo 等超链接地址添加视频。
  • /audio 从本地或者 SoundCloud, Spotify 等超链接地址添加音频。
  • /code 创建一个代码块，然后你可以复制任何的代码进去。
  • /file 可以上传任意本地文件。
  • /embed 可以从 500+ 的内容中选择并创建。了解更多可以创建的内容→

高级

  • /comment 在 block 的任意位置创建一个评论。
  • /duplicate 重复选中的 block 。
  • /moveto 把选中的 block 移到其它的 page 。
  • /delete 删除选中的 block 。
  • /toc 创建当前页面的目录。
  • /button 或者 /template 创建一个模板按钮，用来重复添加你定义的任何 block 组合。 了解更多与模板相关的内容→
  • /bread 插入一个当前 page 所在位置的面包屑菜单。
  • /math 或者 /latex 允许你使用 TeX 来编写数学方程式和符号。
